# Images Directory

This directory contains visual assets for the VMware Tools Auto-Upgrade solution.

## Files

- `vmware-tools-auto-upgrade-process.png` - Process flow diagram showing the complete workflow
- `sample-output-dryrun.png` - Sample output from dry-run mode execution
- `sample-output-apply.png` - Sample output from apply mode execution

## Usage

Images are referenced in the main README.md file to provide visual context for the solution's functionality and output examples.

## Image Guidelines

- Use PNG format for screenshots and diagrams
- Maintain consistent styling and branding
- Include descriptive alt text for accessibility
- Keep file sizes optimized for web viewing
