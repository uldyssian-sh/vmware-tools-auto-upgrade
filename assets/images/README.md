# Repository Assets

This directory contains images and other assets used in the repository documentation.

## Images

### Screenshots
- `vmtools-before-after.png` - Before/after comparison screenshot
- `powercli-execution.png` - PowerCLI script execution example
- `dry-run-output.png` - Dry-run mode output example

### Diagrams
- `workflow-diagram.png` - Script execution workflow
- `architecture-overview.png` - Solution architecture overview

## Usage Guidelines

### Image Standards
- Use PNG format for screenshots
- Maintain consistent sizing (max width: 800px)
- Include descriptive alt text in documentation
- Optimize images for web display

### Documentation Integration
Images should be referenced in documentation using relative paths:

```markdown
![VMware Tools Configuration](../assets/images/vmtools-before-after.png)
```

### File Naming Convention
- Use lowercase with hyphens for separation
- Include descriptive names indicating content
- Add version numbers if multiple versions exist

## Maintenance

- Review images periodically for accuracy
- Update screenshots when UI changes
- Remove unused images to keep repository clean
- Ensure images don't contain sensitive information

---

**Note**: All images should be optimized and free of sensitive data before committing to the repository.
